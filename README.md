# CampusLib — College Library Management System

A full-stack web app for managing a college library.
Built with React + Tailwind, Node.js + Express, PostgreSQL.

## Roles
| Role       | Access |
|------------|--------|
| Admin      | Manage books, members, reports |
| Librarian  | Checkout, return, fines |
| Student    | Search books, view own loans & fines |

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Auth: JWT

## Features
- Book catalog with search
- Member management
- Book checkout & return
- Overdue tracking & fines
- Student self-service portal
- Reports & dashboard
